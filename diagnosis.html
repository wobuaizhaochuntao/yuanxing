🩺 GMV 诊断归因流程 + 规则引擎

GMV 低于预期？
⬇

第一层：流量问题？

  • 场观 < 均值50%？
  • 进房率 < 均值？
→ 封面/标题问题 → 先解决流量

第二层：承接问题？

  • 停留时长 < 均值？
  • 在线持续下滑？
→ 内容/节奏问题 → 优化开场话术

第三层：商品问题？

  • 点击率 < 均值？
  • 成交率 < 均值？
  • 退款率 > 均值？
→ 商品/价格/信任问题
⬇

第四层：转化问题？

  • 下单率/支付率 < 均值？
  • 客单价 < 均值？
→ 临门一脚问题 → 优化话术/商品组合

🔧 诊断规则引擎（核心逻辑）

IF 商品点击率
< 近7场均值 × 0.8
THEN 标记：商品点击率偏低
可能原因：商品讲解深度不足 / 商品卡位置靠下 / 互动过多
建议行动：优化卖点提炼 + 核心商品卡前置 + 聚焦商品讲解
预期提升：5% - 15%
IF 点击成交率
< 近7场均值 × 0.7
THEN 标记：价格或详情页转化有问题
可能原因：价格敏感度高 / 详情页信息不足 / 信任度不够
建议行动：优化价格策略 + 强化售后承诺 + 增加信任背书
预期提升：10% - 20%
IF 人均停留时长
< 近7场均值 × 0.8
THEN 标记：直播间承接能力不足
可能原因：开场话术不够吸引 / 节奏拖沓 / 互动不足
建议行动：优化开场钩子 + 增加互动频率 + 优化节奏
预期提升：8% - 15%
